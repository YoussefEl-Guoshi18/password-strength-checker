# Password Strength Checker

A simple Python program that checks how strong your password is. 
It looks for letters, numbers, and special characters, and gives feedback to make your password stronger.

## How to Use

1. Clone the repository
2. Go into the folder
3. Run the program
4.  Enter your password when prompted. The program will tell you its strength and give suggestions if needed.

## Features

- Checks for letters, digits, and special characters
- Minimum length: 8 characters
- Shows a strength score and visual bar
- Gives suggestions to improve your password
- Lets you try multiple passwords in one session
